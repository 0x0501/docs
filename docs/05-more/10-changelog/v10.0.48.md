# DesireCore v10.0.48

**发布日期**: 2026-05-08

![v10.0.48 更新概览](/img/changelog/v10.0.48.png)

## 新功能
- **心跳通知文件链接**：智能体在心跳通知中现在可以附带生成的文件，您可直接点击打开，Markdown 文件会使用 SuperDoc 预览，其他文件则调用系统默认应用打开。
- **工作流校验增强**：创建或编辑工作流时，系统会更早拦截重复节点、不可达节点、非法分支等运行时不支持的结构，减少“保存能过但执行出错”的情况。

## 修复
- 修复了心跳任务在多智能体并发运行时可能互相影响的问题，让多个智能体的定时任务执行更稳定、不易串扰。

## 改进
- 心跳通知与历史记录现在会保留摘要和文件保存路径，方便您后续回看结果并快速找到产出文件。
- 心跳机制优化为“有结果就通知、确认无变化才静默”，同时支持更合理的超时与工具调用控制，减少无效提醒并提升任务可靠性。
- 工作流 dry-run 预览现在会更清晰展示分支条件、回退路径和引用变量，帮助您在真正运行前更容易看懂执行路径。
