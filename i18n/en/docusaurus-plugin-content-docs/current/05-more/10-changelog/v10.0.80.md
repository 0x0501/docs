# DesireCore v10.0.80

**Release Date**: 2026-06-28

## Fixes
- Fixed an issue where the same message would be displayed repeatedly after restarting the client following a message retry.
- Fixed an issue where the local user data directory could be unexpectedly recreated during login, logout, or session expiry, causing user identity instability.

## Improvements
- When retrying or editing a message, any incomplete previous requests are now automatically cancelled, reducing unnecessary API quota consumption.
