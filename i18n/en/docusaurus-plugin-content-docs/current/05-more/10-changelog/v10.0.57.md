# DesireCore v10.0.57

**Release Date**: 2026-05-14

![v10.0.57 更新概览](/img/changelog/v10.0.57-en.png)

## Fixes
- Fixed an issue on Windows where upgrades to DesireCore could fail for some users when system security scans temporarily locked program files. The upgrade process is now more stable.
- Fixed an issue in some Windows environments where the installer did not fully terminate processes from the previous version, reducing cases where the upgrade would report a failure or be interrupted.

## Improvements
- Improved the compatibility and fault tolerance of the Windows installer, so the installation process can continue as much as possible even if the system lacks certain PowerShell capabilities or command execution encounters errors.
