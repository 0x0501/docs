# DesireCore v10.0.77

**发布日期**: 2026-06-28

![v10.0.77 更新概览](/img/changelog/v10.0.77.png)

## 新功能
- **对话记录导出**：支持将对话导出为 Markdown 或 PDF 文件，可多选部分消息导出，提供工具调用展示模式、是否包含思考过程等高级选项
- **ChatGPT 订阅接入**：可复用本地 Codex CLI 的 ChatGPT 订阅授权，无需 API Key 即可使用 GPT 系列模型
- **算力 Credit 购买**：支持购买算力 credit 套餐包，支持支付宝和微信支付，账户余额和临期额度一目了然
- **最低版本过期引导**：当客户端版本低于官方最低要求时，弹出全屏提示引导升级，确保使用安全版本
- **PDF 逐页阅读增强**：扫描型 PDF 的图片页可直接交给视觉模型理解，长文档支持断点续读和目录查看，混合型文档自动跳过图片页继续读取文字
- **BeautifyImage 图像美化工具**：Agent 可自动去除图片四周的黑边、白边或大片留白，并支持均匀居中补边
- **WebSearch / UnderstandImage 可配置 API**：支持为搜索和图像理解工具配置自定义的外部 API 服务商（如 Tavily、Serper 等）
- **应用与服务目录完善**：派生服务可点击查看详情，支持 HTTP 服务的一键可达性测试和手动重新派生服务
- **团队浮层增强**：点击 TL 徽章可查看团队成员列表与角色，支持从浮层快捷移除成员或进入聊天，团队设置入口更便捷
- **Linux AppImage 安装脚本**：提供一键安装脚本，自动创建命令行包装器和桌面快捷方式

## 修复
- 修复「清除本地数据」在开发模式下实际未删除数据的问题
- 修复 Agent 委派后可能出现大量重复文本导致的卡顿问题
- 修复导入智能体或团队后侧边栏列表不自动刷新的问题
- 修复带文件的消息重试后文件错误残留到输入框上方的问题
- 修复编辑用户消息时气泡宽度塌缩、文字显示不全的问题
- 修复克隆智能体时仅勾选关系或根级偏好导致操作失败的问题
- 修复 Agent 自助注册服务在打包版本中不可用的问题
- 修复 DeepSeek 等模型通过代理访问时因角色不兼容导致报错的问题
- 修复开发模式重启后偶尔出现审批卡片不弹出的问题
- 修复应用与服务中 HTTP 服务可达性测试对官方服务报内部错误的问题
- 修复技能列表名称不跟随界面语言切换的问题
- 修复主控智能体可被克隆的问题（已禁止克隆系统核心智能体）

## 改进
- 技能列表的名称和描述现在跟随界面语言实时切换，切换语言后无需手动刷新
- 团队设置面板中隔离模式、远程仓库、Git 等高级功能改为常驻分区，无需特殊操作即可访问
- 应用与服务目录的待审批服务数量支持实时通知更新，无需手动刷新
- 技能 Bash 命令中可通过环境变量访问用户根目录
- 最低版本重新检测在网络异常时不再错误关闭过期提示，避免绕过版本限制
