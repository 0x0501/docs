# DesireCore v10.0.83

**Release Date**: 2026-07-11

## New Features
- **Claude Subscription Integration**: Supports three methods to connect Claude Pro/Max subscriptions as a compute source: native login via Claude Code, in-app OAuth, or manual token. Disabled by default; must be enabled manually.
- **Thinking Depth Control**: Added a depth selector button in the input area (Auto / Off / Shallow / Medium / Deep / Deepest), compatible with both OpenAI and Anthropic formats. Depth setting persists across sessions.
- **Three-tier Prompt System & Prompt Center**: Introduces three layers: Global (user-wide rules), Team (distributed via team Git), and Agent (persona/principles) for layered injection. A centralized "Prompt Center" in the resource manager provides unified editing.
- **Data Migration to New Device**: Added a migration wizard (export from old device → import on new device) and encrypted API key migration. Supports "Merge into current identity" mode, remapping data within the package by identity and merging into the local machine.
- **Windows GUI Automation**: The installer bundles the CUA Driver out of the box; agents can operate native Windows applications (screenshot / click / input / read windows) — zero installation required, ready to go.
- **Built-in Python / Node.js Runtimes**: The installer includes offline archives of Python 3.13 and Node.js 24 LTS. No internet connection needed on first launch to import the runtime environments; the version list supports offline fallback.
- **Compute Preset Update Review**: When official preset compute configurations change, a review panel pops up to compare differences item by item. Users can choose "Adopt upstream" or "Keep mine" per field.
- **Video Generation Upgrade**: Added support for the Doubao Seedance 2.0 video model; the video generation pipeline has been upgraded to the OpenAI Video API standard path.
- **Compute Billing & Announcement Display**: Announcements from providers are shown at the top of the compute selection panel. Models are categorized by text / image / video, with prices displayed as token multipliers or credits per unit.
- **File “Open With” Menu**: File entries in agent replies now have an "Open with" dropdown — open in-app, with the system default app, choose another app, or reveal in file manager.
- **Plan Discount Display**: Plan cards now show the original price with a strikethrough and the discount percentage (e.g. "Save X%" or "Limited Free").
- **Compute Preset Version Display**: The compute configuration page header now shows the current preset data version number, making it easy to check sync status.
- **Manual Refresh for Apps & Services**: Added a manual refresh button on the Apps & Services page.
- **Process Lock**: Strict single-instance execution per data directory; when a second instance is launched, it queues and waits instead of immediately throwing an error, preventing data corruption.
- **agent-service Standalone Deployment**: The Agent service can be deployed as an independent server process, detached from the Electron desktop client, supporting remote single-tenant containerization.

## Fixes
- **Delegation Concurrency Data Consistency**: Fixed multiple concurrency race conditions during parallel delegation: directory conflicts, missing members, count drift, partial file writes, and more.
- **Scheduled Tasks Reply Only, No Execution**: Relaxed the task execution round limit from 1 round to the same maximum as normal conversations, fixing the issue where the agent only replied once and then stopped. Added failure backoff and alert mechanisms.
- **Context Overflow Self-Adaptation**: The system now learns the model’s real context window and output limit from errors, producing large outputs in one go and automatically downgrading/retrying on overflow.
- **Thinking Depth Budget Collapse**: Fixed a bug where 5 depth levels only had 3 actual budget values, making "Deep" and "Deepest" indistinguishable. Completed persistence for depth levels.
- **Cursor Invisible After Newline in Input Box**: After Shift+Enter, accepting predictions, history recall, and other programmatic text changes, the cursor now correctly scrolls into view.
- **Message List Flickering Entire Window**: Fixed the issue where scrolling to the bottom with a touchpad/scroll wheel and continuing to scroll caused scroll chain bubbling that made the entire window flicker.
- **Short Message Action Bar Overflow**: Fixed the problem where action buttons overflowed and obscured the timestamp on short messages.
- **file:// Link Crash**: Fixed a crash when clicking local file links in agent replies, which caused a macOS error or Windows gray screen freeze. Files now open with the system default application.
- **Incorrect List Indicator Color**: Fixed the chat list status light remaining green (idle) when the agent was actively using a tool (e.g., Read); now correctly shows yellow (busy).
- **Market Pointer Skills Invisible**: Fixed the issue where remotely indexed skills (pointer entries) could not be browsed, searched, or opened in the marketplace.
- **Relationship Graph Missing “Deliver” Edge**: Fixed the relationship graph not showing a delivery edge after the agent successfully delivered a result.
- **Requests to Same-Name Models Cross-Routing**: Fixed an issue where when both official cloud compute and local user configurations had a model with the same name, requests were mistakenly routed to the local provider.
- **Offline After Import/Clone**: Fixed the issue where agents appeared "offline" in the list after importing or cloning, requiring a restart to come back online.
- **Duplicate Team Creation and Deletion Remnants**: Fixed issues with duplicate teams piling up and teams not being cleaned up after agents were deleted.
- **Windows “Clear Data” Failure**: Fixed EPERM errors when deleting read-only git files and reserved device name files (nul, etc.).
- **Scheduled Task Relative Time Drift**: Fixed an issue where relative times like "after X minutes" in long conversations drifted as the session aged; now injects real-time on every turn.
- **Windows Path Backslash Loss**: Fixed the issue where backslashes in Windows paths were swallowed by Markdown escaping when the agent repeated them.
- **Multi-select Mode Click Not Working**: Fixed the issue where clicking the selection circle in message multi-select mode did not respond.
- **Service Catalog Status Not Updating**: Fixed the issue where the catalog and details did not refresh in real-time after service approval/registration, and the pending approval window could crash during rendering.
- **Scanned Document Degradation and Repetition**: Fixed an issue where non-vision models reading pure scanned PDFs would fall into infinite repetitive output; now it prompts to switch models directly.
- **GLM Array Tool Parameter Crash**: Fixed a crash where GLM and other models output malformed array-type tool parameters, causing the entire conversation session to crash.
- **DeepSeek Compatibility Gateway 400 Self-Healing**: Fixed two types of 400 errors for deepseek-v4 and other models on Anthropic-compatible gateways; now supports automatic learning of true limits and self-healing.
- **Aborted Query Slot Leak**: Fixed a global concurrency slot leak caused by manually stopping a query — 3 consecutive aborts would lock up all queries. Also patched slot leakage in the system sleep path.
- **Default Model Application Failure**: Fixed an issue where missing or mismatched providerId in the default model configuration caused the default to silently drift to another provider.
- **Abnormal Topic Message Count**: Fixed an issue where the message count in topic details showed abnormal values in the billions.
- **Message Area Horizontal Scrollbar**: Fixed the issue where super-long URLs, file paths, or wide tables forced a horizontal scrollbar and cropped content.
- **AgentFS Hot Reload Failure**: Fixed a bug where file watching did not trigger after modifying agent.json, so agent configuration changes were not applied in real time.
- **Configuration Concurrent Write Update Loss**: Fixed the issue where concurrent writes to agent.json lost fields; now uses in-lock atomic writes to ensure integrity.
- **Windows Zip Reinstall Infinite Loop**: Fixed an infinite loop caused by a startup crash when deleting read-only git files during zip reinstallation.
- **Build Platform Compatibility**: Fixed Windows asar path separator differences causing build failures and macOS symlinks being rewritten leading to signing failures.

## Improvements
- **Context Truncation & Token Estimation Optimization**: More accurate token estimation, smarter long-context truncation (cascade truncation + lossless offload), and a streamlined system prompt base.
- **Sub-task Wait Timeout Relaxed**: Sub-task synchronous wait relaxed from 5 minutes to 30 minutes; timeout no longer automatically aborts, leaving the decision to the calling agent to continue or terminate.
- **Compute Provider Version Threshold**: Providers requiring a newer client version are automatically filtered out and will not be mistakenly selected.
- **Compute API Key Security Hardening**: Only the masked key value (first 4 characters + ••••••••) is shown in the frontend; plaintext can be viewed on demand through a separate secure channel. Bash/file tools are prohibited from accessing key files.
- **SSRF Protection Delegated to Agent**: Network access protection can be enabled per Agent in agent.json, hot-effective without restart.
- **AgentFS Rollback Upgraded to Git Snapshots**: Rollback upgraded from file-level copyFile to full-tree git snapshots, making restoration more complete and reliable (including new files and deletion recovery).
- **Bundled Git**: No longer depends on system-installed Git; provides a deterministic version guarantee.
- **Compute Provider Concurrency Load Control**: Maximum concurrent requests and queue limits can now be configured per provider to avoid overwhelming the upstream with burst requests.
- **Top-level Query Global Queuing**: Queries within the same conversation are queued serially; cross-conversation queries run in parallel, with at most 3 top-level tasks executing simultaneously (excess tasks queue and show their position).
- **Export Package Identity Detection**: When importing data across devices, the source identity is detected, and a clear warning is shown if identities do not match.
- **Delegation Prompt Path Rules**: When passing file resources across agents, absolute paths must be used to avoid path resolution failures due to different working directories.
- **Repository Source Save No Longer Stalls**: After saving a marketplace repository source, the dialog no longer waits for the entire sync cycle to complete before closing.
- **Chat Scrollbar Widened**: Message area scrollbar hit zone widened to 10px, making it easier to grab with the mouse.
- **Relationship Heat Thousands Separator**: Heat values now formatted with thousands separators and one decimal place to prevent long floating-point numbers from breaking the card layout.
- **Message List Content Fully Visible**: Long URLs/paths now auto-wrap, wide tables get internal horizontal scrolling; content is no longer cropped.
- **System Prompt Path Backtick Wrapping**: Windows paths are injected as inline code to prevent mis-escaping by the Markdown renderer.
- **Ls Tool Default Direct Execution**: Directory listing tool now defaults to auto-approval; security boundary is delegated to the OS permission layer, only prompting platform-specific authorization guidance when denied.
- **Conversation PDF Export Client-Side Rendering**: PDF export is now rendered locally in the Electron client, reducing server-side dependency.
- **Standalone Remote Auth & Media Hardening**: Completed the full authentication chain and media signed URL support for standalone deployment mode.
