# DesireCore v10.0.83

**发布日期**: 2026-07-12

![v10.0.83 更新概览](/img/changelog/v10.0.83.png)

## 新功能
- **Claude 订阅接入**：支持通过 Claude Code OAuth 登录或手动配置，使用 Claude Pro/Max 订阅算力，模型调用经内置官方 SDK 后端履约，审计链路完整
- **思考深度控制**：聊天输入区新增思考深度档位按钮，支持关闭/自动/深度多档调节，会话独立记忆；覆盖 OpenAI、Anthropic 及国产模型方言
- **三层提示词体系**：新增全局层（个人）和团队层提示词，与 Agent 层叠加注入；新增集中式提示词中心，统一管理三层提示词的编辑与预览
- **迁移到新设备向导**：设置页新增「迁移到新设备」入口，五步向导引导导出全量数据；新机导入时自动识别迁移包，支持身份对比、预备份保护和重启生效
- **归并到当前身份**：导入迁移包时支持「归并到当前身份」模式，将包内源身份的数据定向重映射到本机名下，本机身份不变
- **加密迁移 API 密钥**：迁移时可选择携带自定义 Provider 的 API 密钥，经口令加密后独立存储（原始 secrets 文件永不进包），导入时逐 Provider 决策合并
- **文件「打开方式」下拉**：聊天消息中的文件条目新增打开方式菜单，支持应用内打开、系统默认应用、选择其他应用、在文件管理器中定位
- **Windows GUI 自动化（CUA Driver）**：Windows 安装包内置 cua-driver，Agent 可操控桌面应用（截图、点击、输入、读取窗口），后台操作不抢焦点，用户零安装开箱即用
- **上下文超限透明自愈**：当上下文超出模型真实上限时自动学习真实窗口并透明重试，大输出自动计算动态预算一次成型，无需手动调整
- **视频生成升级**：升级为 OpenAI Video API 标准格式，接入官方云算力图像/视频生成，并适配豆包 Seedance 2.0 视频模型
- **算力预置更新通知**：当 config-center 发布的官方 Provider 或模型配置发生变更时，通过通知角和审阅面板展示冲突对比与变更明细，用户可逐项决定采用或保留
- **内置运行时离线安装**：安装包内置 Python 3.13 和 Node 24 LTS 离线运行时，首启后台静默导入，无需联网即可完成环境配置
- **结构化提示词编辑**：Agent 的 persona 和 principles 支持 L0/L1/L2 分层表单编辑，保存时自动校验格式与 Schema；系统提示词预览面板可查看各段占比并复制全文
- **应用与服务页面手动刷新**：新增刷新按钮，可手动同步注册表目录与待审批队列
- **任务板探测无痕化**：切换会话时不再因探测不存在的任务板而在控制台刷 404 错误日志
- **关系图谱补齐交付边**：Agent 成功交付结果后，关系图谱中正确显示「交付」关系边，与「委派」边形成闭环
- **算力页面显示预置配置版本号**：头部元数据区展示当前预置数据版本，便于排查配置同步问题
- **排队中的查询支持中止**：用户在排队等待期间可随时停止请求，不再被无条件调度执行
- **顶层查询并发上限可配置**：通过环境变量可自定义同时运行的顶层查询数量，交互式查询自动优先于后台任务

## 修复
- **聊天输入框换行光标不跟随**：按 Shift+Enter 换行、接受预测建议、历史回溯等操作后，光标自动滚入可视区域
- **委派多智能体并发数据错乱**：修复并发委派时团队目录冲突、成员丢失、文件半写损坏、任务计数漂移等数据一致性问题
- **定时任务不执行**：修复定时任务到期后只回复一句确认就被截断、从未真正完成的问题；失败任务正确退避告警而非无限硬刷
- **消息区横向滚动条**：长 URL 和宽 Markdown 表格不再撑出横向滚动条，内容始终完整可见
- **点击 file:// 链接崩溃**：点击 Agent 回复中的本地文件链接不再导致窗口崩溃或灰屏，改为调用系统默认应用打开
- **DeepSeek-v4 两类 400 错误无法自愈**：网关 max_tokens 范围限制和 thinking 块回传缺失两类报错现已自动学习修正并透明重试
- **扫描件遇非视觉模型退化复读**：纯扫描件 PDF 用非视觉模型阅读时，改由系统直接提示切换模型，不再清空工具致模型陷入无限重复输出
- **聊天列表指示灯误导**：智能体执行工具期间列表灯不再错误显示绿色，正确显示黄色忙碌状态
- **市场 Pointer 技能不可见**：ADR-038 指针式技能重新出现在市场列表中，可正常浏览、搜索和查看详情，版本更新检测正确生效
- **默认模型 ProviderId 缺失**：修复历史数据中默认模型配置丢失 ProviderId 导致默认值永久失效的问题
- **credentialSource 毒丸**：修复老客户端合并含未知凭据源的 Provider 预设时整份配置被拒、算力全部失效的问题；现在仅跳过不支持的 Provider，其余正常使用
- **中止查询致全局并发槽泄漏**：多次手动停止查询后不再耗尽全局并发槽导致所有 Agent 卡死，同时修复系统休眠和会话重置两条泄漏路径
- **同名模型请求串台**：当多个 Provider 配置了同名模型时，请求不再错误路由到无关 Provider
- **导入智能体列表离线**：导入或克隆智能体后立即在列表中显示在线状态，不再需要重启应用
- **Fan-out 委派目标校验缺失**：同时委派多个目标时，无效目标现在会被明确报告失败而非静默跳过，委派卡片正确显示所有目标名称
- **AgentFS 热更新失效**：直接编辑 agent.json 等文件后变更即时生效，不再依赖手动刷新
- **send_message 异步回执错配**：并发的异步消息回执不再被错误分配给其他等待的同步请求
- **via-messaging 并发请求回复错配**：同一调用方对同一目标并发发起多个委派/消息/协助请求时，回复精确匹配对应请求
- **聊天滑到底继续滑动整窗闪烁**：消息列表滚到最底部后继续滑动不再触发窗口闪烁
- **聊天消息操作栏溢出**：短消息下方操作按钮不再溢出盖住时间戳
- **多选模式下点击选择圆圈无响应**：修复选择圆圈在某些场景下点击无反应的问题
- **市场仓库源保存卡顿**：添加新仓库源后不再长时间卡在「保存中」或重复显示
- **关系热度数值溢出**：智能体列表的关系热度显示为千分位加一位小数，不再撑破卡片
- **流水线组装阶段无法取消**：获槽后、正式查询前的工具装配与上下文准备阶段现在可响应停止操作
- **服务目录渲染崩溃**：智能体代注册服务时产生的脏数据不再导致待审批页面崩溃，目录状态实时更新
- **团队同名重复创建**：AI 重复调用创建团队时自动去重，不再产生冗余团队
- **删除智能体后团队残留**：被删智能体所在的团队自动解散或移除成员，不再有孤儿团队
- **扫描件遇非视觉模型时清空工具导致退化复读**：现在直接提示切换模型，不再让模型陷入无工具可调的死循环
- **Provider 并发排队无超时无取消**：排队等待期间断开连接或中止请求可立即出队，不再被阻塞
- **Windows 下清除数据失败**：修复只读 git 文件和保留设备名文件导致清除数据报错的问题
- **Windows git 操作失败**：修复 git 二进制路径含空格或冒号时市场同步、Agent 快照等全部失败的问题
- **流水线组装阶段取消检查点缺失**：MCP 工具发现等长耗时步骤期间可响应取消
- **Claude SDK 后端 openai 入口缺失**：OpenAI 格式请求也能正确路由到 Claude SDK 后端

## 改进
- **上下文裁剪与 Token 估算优化**：裁剪采用 cheapest-first 级联策略，可重取工具结果无损卸载；Token 估算覆盖此前漏数的 thinking 和工具名，非 OpenAI 家族模型自动增加余量
- **Provider 并发限流**：支持为每个 Provider 设置最大并发请求数和排队上限，超出部分排队等待而非直接报错，配置即时生效
- **聊天滚动条加宽命中区**：消息列表滚动条轨道加宽至 10px，鼠标更容易拖拽
- **子任务同步等待超时放宽**：委派、spawn_agent、send_message 等子任务的默认等待时间从 5 分钟延长至 30 分钟，超时不再自动中止，改为提示调用方决策
- **Ls 工具默认免审批**：列出目录改为直接执行，macOS 首次访问受保护目录时自动弹出系统授权框，权限被拒时返回引导文案
- **SSRF 防护下沉到逐 Agent 可控**：每个 Agent 可单独开启内网防护，DNS Rebinding 攻击也被阻断
- **消息多选圆圈无障碍优化**：选择圆圈改为语义化按钮，支持键盘操作和屏幕阅读器
- **WorkflowRun 等待超时放宽**：执行工作流时不再被固定超时截断，支持传入自定义超时
- **可重取工具声明式标记**：工具的「可重取」属性改为注册表声明，MCP 工具根据协议自动推导
- **记忆消息范围修复**：话题详情中消息条数不再显示亿级异常数值，历史脏数据自动修复
- **云模型按类型分区显示**：官方算力模型按文本/图片生成/视频生成分区展示，支持按计费模式显示价格，顶部展示供应商公告
- **系统提示 Windows 路径保护**：系统提示词中的 Windows 路径用行内代码包裹，避免反斜杠被 Markdown 转义丢失
- **Persona/Principles 读写性能优化**：基于文件修改时间的缓存，仅在外部编辑后重新解析
- **智能体配置算力能力**：Agent 可帮用户配置 Provider 和 API 密钥，密钥对 Agent 只写不读
- **迁移包容错增强**：旧版导出包和未来版本包均可安全导入，跨身份导入时主动告警
- **委托提示词新增路径规则**：跨 Agent 传递文件资源时自动使用绝对路径
- **算力密钥安全加固**：密钥在 HTTP 响应中自动脱敏，明文查看需 IPC 通道授权；Read/Grep/Ls 工具精确拒绝读取密钥文件
