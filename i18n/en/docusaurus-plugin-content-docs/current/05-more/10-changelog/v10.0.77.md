# DesireCore v10.0.77

**Release Date**: 2026-06-28

![v10.0.77 更新概览](/img/changelog/v10.0.77-en.png)

## New Features
- **Conversation Export**: Export conversations as Markdown or PDF files. You can select multiple messages for export and access advanced options such as tool call display mode and whether to include the thinking process.
- **ChatGPT Subscription Integration**: Reuse the ChatGPT subscription authorization from your local Codex CLI to use GPT series models without an API Key.
- **Compute Credit Purchase**: Purchase compute credit packages, supporting Alipay and WeChat Pay. Account balance and expiring credits are clearly visible at a glance.
- **Minimum Version Expiration Guidance**: When the client version falls below the official minimum requirement, a full-screen prompt guides you to upgrade, ensuring you stay on a secure version.
- **Enhanced PDF Page-by-Page Reading**: Image pages of scanned PDFs can be directly interpreted by the vision model. Long documents support resuming from breakpoints and viewing the table of contents. Mixed-type documents automatically skip image pages and continue reading text.
- **BeautifyImage Image Beautifier**: The Agent can automatically remove black edges, white edges, or large margins around images and supports adding evenly centered padding.
- **Configurable APIs for WebSearch / UnderstandImage**: Configure custom external API providers (e.g., Tavily, Serper, etc.) for search and image understanding tools.
- **App & Service Directory Improvements**: Derived services can be clicked to view details. Supports one-click reachability tests for HTTP services and manual re-derivation of services.
- **Team Flyout Enhancements**: Click the TL badge to view the team member list and roles. Quickly remove members from the flyout or enter a chat. The team settings entry is now more convenient.
- **Linux AppImage Installation Script**: Provides a one-click installation script that automatically creates a command-line wrapper and desktop shortcut.

## Fixes
- Fixed an issue where "Clear local data" did not actually delete data in development mode.
- Fixed an issue where a large amount of duplicate text could appear after Agent delegation, causing lag.
- Fixed an issue where the sidebar list did not refresh automatically after importing an agent or team.
- Fixed an issue where files attached to a message would incorrectly remain above the input box after retrying the message.
- Fixed an issue where the bubble width would collapse and text was cut off when editing a user message.
- Fixed an issue where cloning an agent would fail if only relationships or root preferences were selected.
- Fixed an issue where the agent self-registration service was unavailable in the packaged build.
- Fixed an issue where models like DeepSeek would throw errors when accessed via proxy due to role incompatibility.
- Fixed an issue where approval cards occasionally did not pop up after restarting in development mode.
- Fixed an issue where HTTP service reachability tests in Apps & Services reported internal errors for official services.
- Fixed an issue where skill list names did not update when the interface language was switched.
- Fixed an issue where the main controller agent could be cloned (cloning of system core agents is now disabled).

## Improvements
- Skill list names and descriptions now switch in real time with the interface language, eliminating the need for manual refresh after a language switch.
- Advanced features such as isolation mode, remote repository, and Git in the team settings panel are now permanently displayed in a dedicated section, accessible without any special actions.
- The pending service count in the Apps & Services directory now updates with real-time notifications, so no manual refresh is needed.
- Bash commands in skills can now access the user’s home directory via an environment variable.
- The minimum version re-check no longer incorrectly closes the expiration prompt when the network is unavailable, preventing bypass of version restrictions.
