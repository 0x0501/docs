# 财务助手

## 真实工作流

财务场景真实存在于“制度明确但人工核对很繁琐”的工作里，例如预算草案、费用审核、成本归因和报表整理。

| 维度 | 真实情况 |
|------|----------|
| 触发点 | 月末报销审核、季度预算编制、费用超支复盘、部门成本分析 |
| 现有材料 | 历史预算、实际支出、业务计划、报销制度、发票和审批记录 |
| 卡点 | 制度条款细，单据多，超标原因和缺失材料需要逐条整理 |
| DesireCore 介入 | 财务助手按制度核对费用，汇总超标项、缺票项、预算差异和调整建议 |
| 验收结果 | 财务人员拿到预算草案或审核报告，再处理例外情况和审批意见 |

## 推荐智能体

**财务助手** -- 适合预算测算、费用制度核对、报表归档、成本分析和异常费用提示。

## 前期教学：让智能体掌握你的财务制度

> 每家公司的报销标准都不一样。先教清楚公司制度，后续审核才会按你的规则执行，而不是按通用经验判断。

### 教规则

```
你: 我把我们公司的费用报销管理制度传给你学习。
    [上传 公司费用报销制度2024版.pdf]
    有几条特别注意的：
    1. 市内交通：只接受打车软件电子发票，不接受手撕出租车发票（强制）
    2. 餐费：工作日不超过 80 元/人，接待客户不超过 200 元/人（强制）
    3. 出差住宿：北上广深不超过 500 元/晚，其他城市不超过 350 元/晚（强制）
    4. 超标报销：需要附书面说明，经部门主管审批后才能报（偏好）

财务助手: 📝 学到了新内容
          来源：公司费用报销制度2024版.pdf + 本次补充说明
          类型：规则（费用报销标准）
          层级：强制 × 3 项，偏好 × 1 项

          我的理解：
          审核报销单时，交通费只认打车软件电子发票；餐费按工作日
          80 元/人、客户接待 200 元/人把控；住宿按城市等级分档
          （一线 500、其他 350）。超标需附说明并有主管签字。

          [✓ 正确]  [✎ 修改]  [✗ 撤销]

你: [✓ 正确]
```

教学完成后，财务助手已掌握你公司的报销标准。之后提供报销单时，它会按制度做初审。

## 完整对话示例

```
你: 帮我编制下季度销售部门的预算。上季度预算总额 180 万，
    实际支出 182 万。下季度销售目标增长 20%，
    计划新招 2 名销售。

财务助手: 好的，我来编制下季度预算。

        历史数据参考（上季度）
        | 费用类别 | 预算  | 实际  | 执行率  |
        |---------|------|------|---------|
        | 人员成本 | 80 万 | 78 万 | 97.5%  |
        | 差旅费  | 25 万 | 32 万 | 128%   |
        | 业务招待 | 15 万 | 18 万 | 120%   |
        | 市场推广 | 50 万 | 45 万 | 90%    |
        | 办公费用 | 10 万 | 9 万  | 90%    |

        下季度预算草案：

        1. 人员成本：95 万
           - 现有团队：78 万
           - 新增 2 人（2 个月薪资）：12 万
           - 绩效奖金：5 万

        2. 差旅费：40 万
           - 说明：上季度超支 28%，本季度有 3 场展会

        3. 业务招待：20 万
        4. 市场推广：60 万（配合销售目标增长）
        5. 办公费用：12 万

        预算总额：227 万（环比 +24.7%）

        风险提示
        差旅费增长 60%，建议设置月度预算预警线。

        需要我生成正式的预算申请表吗？

你: 差旅费能控制在 35 万吗？另外帮我审核这几张报销单。

财务助手: 差旅费调整为 35 万。削减建议：
        - 部分远程会议替代出差，节约 3 万
        - 展会住宿选择经济型酒店，节约 2 万

        报销审核结果：
        通过 12 笔 | 退回 2 笔 | 拒绝 1 笔
        详细说明...

        📋 回执摘要
        引用的规则：餐费标准 · 住宿标准 · 交通发票要求
        均来自你的教学（上周）✓
```

## 关键步骤

1. **明确核算对象** -- 先说明部门、项目、期间、预算科目和审批口径
2. **对齐预算与实际** -- 对比预算、实际、预测和历史同期，先找出主要差异
3. **解释差异原因** -- 把差异拆成业务增长、价格变化、一次性支出、超标费用或口径问题
4. **审核制度符合性** -- 对报销单据检查发票类型、金额上限、审批链和补充说明
5. **输出审批建议** -- 按通过、退回补材料、超标需审批、拒绝报销分类汇总

## 最终成果

一份预算草案或费用审核报告，包含预算-实际差异、测算依据、超标项、缺失材料和待确认事项。

:::warning 免责声明
财务助手提供的是辅助整理和初步分析，不构成专业财务、审计或税务意见。涉及重大财务决策、纳税申报和对外披露时，应由专业人员确认。
:::

:::tip 最佳实践
- 编制预算时提供尽可能完整的历史数据，预算准确性会显著提高
- 费用审核时可以配置你公司的报销标准（如差旅标准、招待限额），智能体会严格按照标准审核
- 月末、季末可以让财务助手整理财务报表，汇总关键指标和异常预警
- 月末批量审核时，可以让财务助手先汇总超标项、缺票项和待补充说明，再由财务人员做最终处理
- 财务分析不要只看差异金额，还要解释差异性质：临时支出、结构变化、收入增长带动，还是预算编制偏差
:::
