# DesireCore v10.0.83

**发布日期**: 2026-07-12

## 新功能
- **Windows GUI 自动化（CUA Driver）**：Windows 客户端内置 cua-driver，智能体可直接操作原生应用（截图、点击、输入、读取窗口状态），开箱即用无需额外安装
- **Claude 订阅接入**：支持 Claude Pro/Max 订阅作为算力来源，提供本机 Claude Code 登录、应用内 OAuth 和手动 setup-token 三种凭证方式
- **三层提示词体系与集中式提示词中心**：新增全局层、团队层和智能体层提示词叠加注入，可通过统一提示词中心编辑和管理
- **系统提示词管理增强**：智能体 persona/principles 支持结构化表单编辑，新增系统提示词预览面板（含 section 可视化和能力声明开关）
- **思考深度 UI 控制**：输入区新增思考档位选择器（7 档），适配 OpenAI / Anthropic 双格式及多厂商方言
- **内置离线运行时**：安装包内置 Python 3.13 和 Node.js 24 LTS，首启离线可用，版本列表联网失败时降级至内置清单
- **算力预置合并冲突保护**：您手动修改的 Provider 字段不会被上游覆盖，上游变更时会收到通知并逐项审阅
- **数据备份与迁移体系**：新增「迁移到新设备」导出向导、导入身份识别与归并、加密迁移 API 密钥
- **视频生成升级**：视频生成升级为 OpenAI Video API 协议，接入官方云算力，支持豆包 Seedance 2.0
- **文件「打开方式」下拉**：对话中的文件条目可自选打开方式（应用内 / 默认应用 / 其他应用 / 在文件夹中显示）
- **对话 PDF 导出改为客户端渲染**：对话导出 PDF 改为客户端直接渲染，支持远程部署场景
- **算力配置版本号显示**：算力页面头部新增当前预置配置版本号，方便排查同步问题
- **Provider 并发限流配置**：可在 Provider 配置中设置最大并发请求数、最大排队数及排队超时
- **套餐原价划线与折扣展示**：支付页套餐卡片支持显示划线原价和折扣信息
- **Agent 文件回撤升级为 Git 全树快照**：智能体文件回撤更完整，支持更多还原场景
- **服务目录手动刷新**：「应用与服务」页面新增刷新按钮，可手动同步服务目录
- **上下文超限自动恢复与动态输出预算**：模型超上下文时自动学习真实窗口并透明重试，大输出任务尽量一次完成
- **关系图谱补齐交付边**：成功交付结果时，关系图谱正确显示「交付」关系边

## 修复
- **Ls 目录列表免审批执行**：列出目录内容直接执行，无需审批；系统权限不足时给出清晰的授权引导
- **智能体列表重复团队**：修复 AI 重复创建同名团队导致列表出现多个相同团队的问题
- **智能体导入后状态离线**：导入或克隆的智能体不再显示为「离线」
- **委派多智能体并发数据正确性**：修复并行委派时可能出现的数据丢失、文件损坏和目录冲突
- **列表状态指示灯求真**：智能体使用工具时正确显示黄色（忙碌），而非绿色（在线）
- **消息滚动条加宽**：聊天区滚动条命中区从 4px 加宽到 10px，拖拽更方便
- **长内容完整可见**：超长 URL / 路径 / 宽表格不再出现横向滚动条，内容自动折行或内部横滚
- **短消息操作栏不再遮盖时间戳**：按钮在窄气泡下自动换行，不再盖住时间戳
- **消息列表滚动到底不再闪烁**：修复滑到底后继续滑动导致整窗闪烁的问题
- **定时任务真正执行**：到期后不再只回一句就被截断，可完成多步工具调用
- **定时任务失败不再无限硬刷**：任务失败后正确执行指数退避，不再无限重试
- **长时间会话时间漂移修复**：Agent 的“当前时间”不再停留在会话开始时刻
- **扫描件遇非视觉模型不再复读崩溃**：直接提示切换视觉模型，不再进入无限重复输出
- **file:// 链接不再崩溃**：点击本地文件链接改为系统默认应用打开，不再报错或卡死
- **Windows git 路径含空格不再失败**：修复 git 在 Program Files 等路径下所有操作失败的问题
- **服务目录状态实时更新**：审批或注册后目录自动刷新，待审批面板不再因脏数据崩溃
- **Windows 清除数据不再失败**：修复因只读 git 文件和保留设备名文件导致清除失败的问题
- **同名模型路由串台修复**：官方云算力与本地配置有同名模型时不再错误路由
- **思考深度档位预算塌缩修复**：5 个档位恢复正确的预算值，最深档位可达 32K+
- **输入框换行后光标自动滚入可视区**：Shift+回车、接受预测等操作后光标始终可见
- **委派卡片正确显示目标名**：fan-out 委派不再显示为“unknown”
- **并发消息回复匹配修复**：同一发起方对同一目标并发多个消息时不再错配回复
- **排队中的查询可中止**：排队中点停止按钮会立即出队，不再被调度执行
- **Provider 并发限流完善**：限流器统一识别 Provider ID，排队时向用户反馈状态，支持诊断查看
- **DeepSeek-V4 部分 400 错误自愈**：修复 desirecore-cloud 上 max_tokens 范围和 thinking 回传两类 400 无法自愈的问题
- **Windows 路径反斜杠丢失修复**：系统提示中的路径加行内代码包裹，显示和复制不再丢反斜杠
- **市场中 pointer 技能恢复**：被错误过滤的 20 个指针型技能回归市场
- **macOS 公证修复**：修复 macOS 构建因内置 Python 签名问题被公证拒绝

## 改进
- **运行时环境开箱即用**：工具链目录统一到 `~/.desirecore/runtime`，老用户升级自动完成一次性的目录迁移
- **独立部署模式**：agent-service 支持脱离桌面客户端作为独立服务器运行（单租户、远程安全访问、一客户一容器）
- **Claude SDK 后端增强**：支持请求排队限流、冷启动耗时观测、登录取消按钮、Session 续接和 WebSearch
- **多并发治理全面加固**：进程锁防多开、全局并发槽排队（可中止、支持优先级）、Provider 级信号量和写路径全面原子化
- **Agent 能力边界紧固**：Claude Agent SDK 只承担大模型请求，工具、技能、权限和文件执行全部由 DesireCore 治理链控制
- **系统提示词精简**：去除重复自我认知和废弃能力声明，节省 token 消耗
- **任务板探测静默化**：切换会话时不再在浏览器控制台输出 404 错误日志
- **子任务超时放宽**：delegate / spawn_agent / send_message 等待超时从 5 分钟放宽至 30 分钟，超时后不强制中止
- **数据导入导出可靠性增强**：修复 zip slip 漏洞和导入失败后服务停摆，增加 manifest 和身份边界检测
- **credentialSource 防毒丸**：改为开放 string，未知值不再导致配置崩溃；启动时损坏配置自动降级重建
- **standalone 鉴权加固**：统一请求鉴权入口，新增媒体签名 URL，保障远程场景下图片与视频访问
- **开发与稳定性改进**：修复 worktree 开发模式首屏白屏、休眠中断会话恢复、流式请求断流等多项问题
