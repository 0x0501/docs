# DesireCore v10.0.83

**发布日期**: 2026-07-11

## 新功能
- **Claude 订阅接入**：支持通过 Claude Code 本机登录、应用内 OAuth 或手动 token 三种方式接入 Claude Pro/Max 订阅作为算力来源，默认关闭需主动开启
- **思考深度控制**：输入区新增思考档位选择按钮（自动/关闭/浅/中/深/最深），兼容 OpenAI 与 Anthropic 双格式，档位设置跨会话持久化
- **三层提示词体系与提示词中心**：新增全局层（用户通用规则）、团队层（随团队 Git 分发）和 Agent 层（persona/principles）三层叠加注入；资源管理器新增集中式「提示词中心」统一编辑
- **数据迁移到新设备**：新增迁移向导（旧机导出 → 新机导入）与 API 密钥加密迁移；支持「归并到当前身份」模式，将包内数据按身份重映射后并入本机
- **Windows GUI 自动化**：安装包内置 CUA Driver，智能体可操作 Windows 原生应用（截图/点击/输入/读取窗口），零安装开箱即用
- **内置 Python/Node.js 运行时**：安装包内置 Python 3.13 和 Node.js 24 LTS 离线归档，首启无需联网即可导入运行环境，版本列表支持离线兜底
- **算力预置更新审阅**：当官方预置算力配置有变更时，弹出审阅面板逐项对比差异，用户可逐字段选择「采用上游」或「保留我的」
- **视频生成升级**：新增支持豆包 Seedance 2.0 视频模型，视频生成链路升级为 OpenAI Video API 标准路径
- **算力计费与公告展示**：算力选择面板顶部显示供应商公告，模型按文本/图片/视频分区，价格按 token 倍率或 credits/单位 分类展示
- **文件「打开方式」菜单**：智能体回复中的文件条目新增打开方式下拉——应用内打开、系统默认应用、选择其他应用、在文件管理器中定位
- **套餐折扣展示**：套餐卡片支持显示原价划线和折扣比例（如「省 X%」或「限免」）
- **算力预置版本号显示**：算力配置页面头部显示当前预置数据版本号，方便确认同步状态
- **应用与服务手动刷新**：应用与服务页面新增手动刷新按钮
- **进程锁**：同一数据目录严格单实例运行，多开时第二个进程排队等待而非直接报错，防止数据损坏
- **agent-service 独立部署**：Agent 服务可脱离 Electron 桌面客户端，以独立服务器进程部署，支持远程单租户容器化

## 修复
- **委派并发数据一致性**：修复并行委派时目录冲突、成员丢失、计数漂移、文件半写等多项并发竞态
- **定时任务只回复不执行**：任务执行轮次从 1 轮放宽至与普通对话一致的轮次上限，解决 Agent 回复一句即被截断的问题；补齐失败退避与告警机制
- **上下文超限自适应**：系统从报错中学习模型真实上下文窗口与输出上限，大输出一次成型、超限时自动降级重试
- **思考档位预算塌缩**：修复 5 个深度档位实际只有 3 个预算值、「深」与「最深」无法区分的缺陷，补齐档位持久化
- **输入框换行后光标不可见**：Shift+回车、接受预测、历史回溯等编程式改文本后光标正确滚入可视区
- **消息列表整窗闪烁**：修复触控板/滚轮滚到底继续滑动时，滚动链冒泡导致整窗闪烁
- **短消息操作栏溢出**：修复短消息时操作按钮溢出遮盖时间戳的问题
- **file:// 链接崩溃**：修复点击智能体回复中本地文件链接导致 macOS 报错 / Windows 灰屏卡死的崩溃，文件改由系统默认应用打开
- **列表指示灯颜色不准**：修复智能体正在使用工具（如 Read）时，聊天列表状态灯仍为绿色而非黄色（忙碌）的问题
- **市场 pointer 技能不可见**：修复远程索引的技能（pointer 条目）无法在市场中浏览、搜索、打开的缺陷
- **关系图谱缺失「交付」边**：修复智能体成功交付结果后关系图谱不显示交付关系边的问题
- **同名模型请求串台**：修复官方云算力与用户本地配置了同名模型时，请求被错误路由到本地 provider 的问题
- **导入/克隆后列表离线**：修复导入或克隆智能体后列表中显示「离线」、需重启才恢复在线的问题
- **团队重复创建与删除残留**：修复同名团队重复堆积、删除智能体后团队未清理的问题
- **Windows「清除数据」失败**：修复删除只读 git 文件和保留设备名文件（nul 等）时的 EPERM 错误
- **定时任务相对时间漂移**：修复长时间对话中「X 分钟后」等相对时间随会话变旧而漂移的问题，每轮注入实时时间
- **Windows 路径反斜杠丢失**：修复智能体复述 Windows 路径时反斜杠被 Markdown 转义吞掉的问题
- **多选模式点击无效**：修复消息多选模式下点击选择圆圈无响应的问题
- **服务目录状态不更新**：修复服务审批/注册后目录和详情不实时刷新、待审批窗口可能渲染崩溃的问题
- **扫描件退化复读**：修复非视觉模型读取纯扫描件 PDF 时陷入无限重复输出，现改为直接提示切换模型
- **GLM 数组型工具参数崩溃**：修复 glm 等模型输出畸形的数组型工具参数导致整条会话崩溃的问题
- **DeepSeek 兼容网关 400 自愈**：修复 deepseek-v4 等模型在 Anthropic 兼容网关上的两类 400 错误，支持自动学习真实上限并自愈
- **中止查询槽泄漏**：修复手动停止查询后全局并发槽被永久占用——连续 3 次中止即导致所有查询卡死；同步修补系统休眠路径的槽泄漏
- **默认模型生效失败**：修复默认模型配置中 providerId 缺失/错配导致默认值静默漂移到其他 provider 的问题
- **话题消息条数异常**：修复话题详情中消息条数显示亿级异常数值的问题
- **消息区横向滚动条**：修复超长 URL/文件路径/宽表格撑出横向滚动条、内容被裁剪的问题
- **AgentFS 热更新失效**：修复修改 agent.json 后文件监听不触发、智能体配置变更不实时生效的问题
- **配置并发写丢更新**：修复 agent.json 被并发写入时丢字段的问题，新增锁内原子更新保证完整性
- **Windows zip 重装死循环**：修复 zip 重装时因只读 git 文件删除失败导致启动崩溃的死循环
- **构建平台兼容**：修复 Windows asar 路径分隔符不同导致构建失败、macOS 符号链接被重写导致签名失败的问题

## 改进
- **上下文裁剪与 Token 估算优化**：更准确地估算 token、更智能地裁剪超长上下文（级联截断+无损卸载）、系统提示词底座精简
- **子任务等待超时放宽**：子任务同步等待从 5 分钟放宽到 30 分钟，超时不再自动中断，由调用方 Agent 决定继续或终止
- **算力 Provider 版本门槛**：需要更新客户端版本才能使用的 provider 会被自动过滤，不会被误选
- **算力 API Key 安全加固**：前端仅显示密钥掩码值（前 4 位+••••••••），明文通过独立的安全通道按需查看；Bash/文件工具禁止访问密钥文件
- **SSRF 防护下沉到 Agent**：网络访问防护可在 agent.json 中按 Agent 单独开启，热生效无需重启
- **AgentFS 回撤升级为 Git 快照**：回撤从文件级 copyFile 升级为 git 全树快照，恢复更完整可靠（含新建文件和删除恢复）
- **随包内置 Git**：不再依赖系统安装的 Git，提供确定性版本保障
- **算力 Provider 并发负载控制**：可按 Provider 配置最大并发请求数和排队上限，避免瞬时请求打爆上游
- **顶层查询全局排队**：同对话内查询排队串行，跨对话并行，最多同时执行 3 个顶层任务（超出排队等待并显示位次）
- **导出包身份检测**：跨设备导入数据时检测来源身份，身份不匹配时显示明确告警
- **委派提示词路径规则**：跨 Agent 传递文件资源须使用绝对路径，避免工作目录不同导致路径解析失败
- **仓库源保存不再卡顿**：市场仓库源保存后不再等待整轮同步完成才关闭弹窗
- **聊天滚动条加宽**：消息区滚动条命中区加宽至 10px，鼠标拖拽更易命中
- **关系热度千分位展示**：热度数值格式化为千分位+1 位小数，避免长浮点数撑破卡片布局
- **消息列表内容完整可见**：长 URL/路径自动折行、宽表格内部横滚，不再裁剪内容
- **系统提示路径反引号包裹**：Windows 路径以行内代码形式注入，防止被 Markdown 渲染器误转义
- **Ls 工具默认直接执行**：目录列表工具默认免审批，安全边界交由操作系统权限层，仅被拒绝时返回平台专属授权引导
- **对话 PDF 导出客户端渲染**：导出 PDF 改为在 Electron 客户端本地渲染，减少服务端依赖
- **Standalone 远程鉴权与媒体加固**：补齐独立部署模式的完整鉴权链路和媒体签名 URL 支持
