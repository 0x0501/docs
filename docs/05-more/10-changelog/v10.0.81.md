# DesireCore v10.0.81

**发布日期**: 2026-06-28

![v10.0.81 更新概览](/img/changelog/v10.0.81.png)

## 新功能
- **回撤（Rewind）**：支持一键回到对话中任意一条用户消息之前，并恢复当时被智能体修改过的文件状态，原对话分支保留不丢失。
- **文生图/视频工具**：新增 GenerateImage 和 GenerateVideo 两个内置工具，智能体可直接调用 MiniMax、火山 Seedance/Seedream、阿里快乐马等厂商生成图片或视频。
- **任务板**：智能体可将复杂任务拆解为子任务并逐步推进，用户可通过悬浮任务板实时查看任务进度和状态流转。
- **历史版本查看**：在 Markdown 编辑器和资源管理器中新增「历史」按钮，可查看文件的 Git 提交历史记录。
- **运营公告**：智能体列表顶部新增可关闭的运营公告横幅，用于展示系统维护或优惠活动通知。

## 修复
- 修复重试消息后重启应用，被重试的消息会重复显示的问题。
- 修复聊天记录面板遗漏失败、运行中等非空闲状态的会话消息，现在与主聊天区显示一致。
- 修复助手回复中的文件路径在刷新后无法点击打开的问题。
- 修复 AI 生成长内容（如图表）时流意外中断导致会话直接终止、不自动重试的问题。
- 修复智能体对同一工具反复用相同方式调用失败时无限重试、白耗 token 的问题。
- 修复 Draw.io 图表在暗色模式下显示发黑的问题，并修复编辑后保存不生效、发送给 AI 拿到旧图等问题。
- 修复 Mermaid 图表全屏预览时关闭按钮被窗口控件遮挡无法点击的问题。
- 修复 super-doc 升级后 Mermaid 插件初始化导致应用崩溃的问题。
- 修复编辑对话气泡时，智能体的审批或问询卡片被弹窗遮挡无法点击的问题。
- 修复 IMAP 邮箱长连接超时导致应用弹出崩溃对话框的问题。
- 修复网络连接被重置等瞬态错误导致应用弹出原生崩溃对话框的问题。
- 修复对话导出为 Markdown 或 PDF 时无法包含图片，且 PDF 中思考过程块显示为原始代码的问题。
- 修复通知中心审批通知的「批准/拒绝」按钮点击无响应的问题。
- 修复后台登录态失效时客户端不会自动退出并提示重新登录的问题。
- 修复云端登录/登出时本地用户 ID 错误切换，导致用户数据目录创建混乱的问题。
- 修复应用商店安装应用后「已安装」面板为空、操作按钮无效、卸载不清理的问题。
- 修复文生视频结果在消息气泡中宽度溢出、显示不全的问题。

## 改进
- 复制消息按钮现在区分「复制纯文本」（去掉 Markdown 标记）和「复制 Markdown」（保留源码），行为更直观。
- Draw.io 图表卡片改用静态 SVG 渲染，长对话中不再堆积多个常驻编辑器，性能大幅提升。
- 浏览器自动化出错时提供明确的分类诊断和解决建议（如未安装浏览器、调试端口未就绪等），不再笼统报错。
- 应用商店安装进度从固定 43% 改为不确定态动画，安装完成的应用支持启动、停止、重启、更新等操作。
- 文生图/视频工具新增 new-api 网关中转模式，可统一通过 OpenAI 兼容接口调用多厂商媒体生成服务。
