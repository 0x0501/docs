# DesireCore v10.0.74

**发布日期**: 2026-06-06

![v10.0.74 更新概览](/img/changelog/v10.0.74.png)

## 新功能
- **PDF 文档大纲读取**：智能体现在可以先返回 PDF 的章节目录与页码，帮助你在长文档中更快定位需要阅读的部分。
- **文档导出为 PDF**：智能体新增可直接将文档导出为 PDF 的内置能力，无需再依赖临时脚本或额外转换流程。

## 修复
- 修复了 Agent 文档导出 PDF 失效的问题，现在通过相关导出接口可以正常生成 PDF 文件。
- 修复了编辑器导出 PDF 时的格式问题，导出的表格、代码块、标题锚点和页面内容会更完整、更规整。
- 修复了读取超大页或超长内容的 PDF 时可能出现的续读异常与卡住问题，长文档阅读过程更稳定。

## 改进
- 优化了智能体读取长篇 PDF 的速度与稳定性，再次翻读同一文档时更快，减少重复解析带来的等待。
- 改进了 PDF 阅读策略，优先提取文字内容并在达到上下文预算后自动停止无效解析，减少卡顿和无用消耗。
- 优化了图表页、扫描页等复杂 PDF 页面的读取表现，即使当前模型不是视觉模型，也能尽量返回可提取的文字内容。
- 改进了超长 PDF 的处理方式，请求页数过多时不再直接拒绝，而是先读取前 50 页并提示继续读取。
- 优化了需要视觉委派的 PDF 页读取效果，复杂页面的识别结果更完整，重复读取时响应也更快。
- 统一了编辑器导出与 Agent 生成 PDF 的渲染样式，整体导出结果更加一致。
