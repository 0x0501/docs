# DesireCore v10.0.80

**Release Date**: 2026-06-28

![v10.0.80 更新概览](/img/changelog/v10.0.80-en.png)

## Fixes
- Fixed an issue where some messages would reappear after retrying a conversation message and restarting the app.
- Fixed an issue where signing in or out of a cloud account could create duplicate local user data, ensuring consistent user identity.

## Improvements
- When retrying or editing a message, the system now automatically cancels previous background requests, reducing unnecessary computing resource consumption.
