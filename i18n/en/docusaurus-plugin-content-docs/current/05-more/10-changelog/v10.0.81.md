# DesireCore v10.0.81

**Release Date**: 2026-06-28

![v10.0.81 更新概览](/img/changelog/v10.0.81-en.png)

## New Features
- **Rewind**: Jump back to any previous user message in the conversation with a single click, and restore the file states as they were when the agent modified them at that time. The original conversation branch is preserved and not lost.
- **Text-to-Image/Video Tools**: Two new built-in tools, GenerateImage and GenerateVideo, allow the agent to directly generate images or videos via providers such as MiniMax, Volcano Engine Seedance/Seedream, and Alibaba Happy Horse.
- **Task Board**: The agent can break complex tasks into subtasks and work through them step by step. Users can track task progress and status transitions in real time via a floating task board.
- **History Version Viewer**: A "History" button has been added to the Markdown editor and resource explorer, allowing you to view the file’s Git commit history.
- **Operational Announcements**: A dismissible announcement banner now appears at the top of the agent list, used to display system maintenance notices or promotional events.

## Fixes
- Fixed an issue where retried messages would appear duplicated after restarting the app following a message retry.
- Fixed an issue where the chat history panel omitted messages from sessions in non-idle states (e.g., failed, running), now consistent with the main chat area.
- Fixed the issue where file paths in assistant replies could not be clicked to open after a refresh.
- Fixed an issue where generating long content (e.g., charts) with AI would cause the session to terminate abruptly without auto-retry if the stream was unexpectedly interrupted.
- Fixed an issue where the agent would infinitely retry calling the same tool in the same way after failures, wasting tokens.
- Fixed an issue where Draw.io diagrams appeared black in dark mode, and fixed problems where edits were not saved and the AI received the old diagram.
- Fixed an issue where the close button was obscured by window controls and unclickable when previewing Mermaid diagrams in fullscreen.
- Fixed an issue where initializing the Mermaid plugin after upgrading super-doc caused the app to crash.
- Fixed an issue where agent approval or inquiry cards were blocked by pop-ups and unclickable when editing a message bubble.
- Fixed an issue where a long-lived IMAP email connection timeout caused the app to show a crash dialog.
- Fixed an issue where transient errors like network connection resets caused the app to show a native crash dialog.
- Fixed an issue where exporting conversations to Markdown or PDF did not include images, and the thinking process blocks appeared as raw code in PDFs.
- Fixed an issue where the ‘Approve/Reject’ buttons in approval notifications from the notification center were unresponsive.
- Fixed an issue where the client would not automatically log out and prompt for re-login when the background session expired.
- Fixed an issue where local user IDs were incorrectly switched during cloud login/logout, causing confusion in user data directory creation.
- Fixed an issue where after installing apps from the app store, the 'Installed' panel was empty, action buttons were nonfunctional, and uninstall did not clean up.
- Fixed an issue where text-to-video results overflowed horizontally in the message bubble and were not fully displayed.

## Improvements
- The copy message button now differentiates between ‘Copy plain text’ (stripping Markdown markup) and ‘Copy Markdown’ (preserving source), making the behavior more intuitive.
- Draw.io diagram cards now use static SVG rendering; long conversations no longer accumulate multiple persistent editors, greatly improving performance.
- Browser automation errors now provide clear categorized diagnostics and resolution suggestions (e.g., browser not installed, debug port not ready), instead of generic error messages.
- The installation progress in the app store has been changed from a fixed 43% to an indeterminate animation, and installed apps now support launch, stop, restart, and update operations.
- A new-api gateway relay mode has been added to the text-to-image/video tools, allowing unified calls to multiple vendors’ media generation services through an OpenAI-compatible interface.
