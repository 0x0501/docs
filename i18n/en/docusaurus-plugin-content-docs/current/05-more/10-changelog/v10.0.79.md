# DesireCore v10.0.79

**Release Date**: 2026-06-27

![v10.0.79 更新概览](/img/changelog/v10.0.79-en.png)

## New Features
- **Official Compute Model Upgrade**: A comprehensive consumption multiplier badge relative to the default model has been added to the model list. When there are too many model capability tags, they automatically collapse, making cost comparison and model capabilities clear at a glance.
- **Smart Filtering for Conversation Models**: Image and video generation models are automatically disabled and greyed out in the conversation model selector. Hovering over them shows a tooltip explaining why they are unavailable, preventing accidental selection that could cause conversation errors.

## Fixes
- Fixed an issue where upstream large model streaming responses closed the connection prematurely without sending an end marker, causing the application process to crash.
- Fixed an issue where the background of the team remote repository connection dialog was displayed incorrectly and some fields were not translated.
