# DesireCore v10.0.85

**发布日期**: 2026-07-14

![v10.0.85 更新概览](/img/changelog/v10.0.85.png)

## 新功能
- **Windows 升级进度窗**：在 Windows 上点击升级按钮后，现在会弹出独立进度窗口，全程可视化展示安装进度，完成后可直接启动新版本，不再因长时间静默安装而误以为升级失败。
- **公告弹窗登录入口**：升级公告弹窗新增“已有账号？立即登录”链接，已有账号的用户可直达登录界面，无需返回主界面手动操作。

## 修复
- 将 Windows 升级弹窗中的“立即重启”按钮更名为“退出应用并自动升级”，消除因流程不自动重启而造成的文案误导。
- 修复了因残留临时文件导致升级进度窗可能无法弹出、应用退出后亦无任何反应的问题。
- 修复了高分辨率（如 150% 缩放）下，升级进度窗失败提示中的“打开安装目录”按钮文字被截断的问题。
