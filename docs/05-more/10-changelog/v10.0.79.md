# DesireCore v10.0.79

**发布日期**: 2026-06-27

![v10.0.79 更新概览](/img/changelog/v10.0.79.png)

## 新功能
- **官方算力模型升级**：模型列表新增相对默认模型的综合消耗倍率徽章，模型能力标签过多时自动折叠，让成本对比和模型能力一目了然。
- **对话模型智能过滤**：图像与视频生成类模型在对话模型选择器中自动禁用并置灰，鼠标悬停时提示不可选原因，避免误选导致对话异常。

## 修复
- 修复上游大模型流式回复在未发送结束标记时提前关闭连接，导致应用进程崩溃的问题。
- 修复团队远程仓库连接弹窗背景显示异常和部分字段未翻译的问题。
