# DesireCore v10.0.83

**发布日期**: 2026-07-12

## 新功能
- **Claude 订阅接入**：支持使用 Claude Pro/Max 订阅作为算力来源，可通过本机 Claude Code 登录、应用内 OAuth 或手动 Token 三种方式连接，所有模型调用经内置 Agent SDK 履约，完整保留审批与审计链路
- **思考深度控制**：聊天输入区新增思考深度档位选择器（自动/关闭/轻度〜最深），支持按会话切换，并可设为全局默认，兼容 OpenAI 与 Anthropic 两种格式
- **三层提示词体系**：新增全局层（个人通用规则）、团队层（团队共享规则）与 Agent 层提示词，三层叠加注入；资源管理器新增「提示词中心」统一管理各层提示词，支持结构化编辑与预览
- **迁移到新设备**：设置页新增「迁移到新设备」引导向导，可导出全量数据（含可选的加密 API 密钥），在新设备上导入时支持身份替换或归并到当前身份
- **API 密钥迁移**：迁移导出向导新增加密导出 API 密钥选项，使用密码保护与 AES-256-GCM 加密，导入时支持逐 Provider 的替换/合并冲突决策
- **Windows GUI 自动化**：Windows 安装包内置 cua-driver 运行时，开箱即可让 Agent 操作 Windows 桌面应用（截图、点击、输入、读取窗口状态），无需手动安装
- **内置离线运行时**：安装包内置 Python 3.13.9 与 Node.js 24.18 LTS 离线运行环境，首启后台静默导入，联网失败时自动回退离线安装，彻底消灭因网络问题导致的环境异常
- **文件「打开方式」菜单**：聊天中生成或引用的文件新增「打开方式 ▾」下拉菜单，可选择应用内打开、系统默认应用、其他应用、或在文件夹中定位，统一三处文件打开逻辑
- **AgentFS 回撤升级为 git 全树快照**：回撤功能改用 git 原生机制完成文件还原，支持跨多轮的精确回退，安装包内置 git 二进制以保证确定性行为
- **算力预置更新审阅**：config-center 推送新的算力配置预设时，客户端新增可视化审阅面板——逐 Provider 展示字段级差异对比，用户可逐项选择「采用上游」或「保留我的」，附带通知角标提醒
- **单租户独立部署**：Agent Service 支持脱离 Electon 独立部署到服务器，提供 Docker 一客户一容器交付方案，支持远程安全访问（实例 token 鉴权 + 环回网格 + 可配 CORS）
- **上下文超限透明自愈**：自动从模型返回的「上下文超限」错误中学习真实窗口大小，下次对话直接按真实值裁剪，超限发生时自动降输出或真窗口重裁输入，无需用户干预
- **视频生成接入豆包 Seedance 2.0**：视频生成工具新增豆包 Seedance 2.0 模型支持，自动适配渠道格式差异
- **智能体配置算力能力**：新增官方「配置算力」技能，Agent 可为用户配置 API 密钥（只写不读）及模型，前端 Provider 详情页密钥明文改为按需查看
- **队列并发控制**：多个 Agent 同时发消息时，超出并发上限的请求自动排队，提示「排队中（第 N 位）」；用户可随时中止排队中的请求

## 修复
- **委派多智能体并发数据安全**：修复并行委派时可能出现的数据丢失、目录碰撞、计数字段漂移、文件半写损坏等问题，确保任意并发度下行为正确
- **「停止」后并发槽永久泄漏**：修复用户手动停止生成后全局并发槽不释放，累积 3 次中止导致所有 Agent 顶层查询永久卡死的问题；同时修补系统休眠、会话重置两条路径的同根泄漏
- **定时任务不执行/失败硬刷**：修复定时任务到期后只回复一句话就被截断（轮次限制过严）、循环任务失败后无退避持续重试、长任务后时间显示错位等问题
- **市场指针技能不可见**：修复市场中全部 pointer 类型技能无法浏览和搜索的问题；修复上游技能无版本号时更新被永久静默跳过的问题
- **同名模型串台**：修复云端模型与本地 Provider 共享模型名（如 deepseek-v4-pro）时，请求被错误路由到本地 Provider 的问题，覆盖对话、委派、心跳等全部 LLM 调用路径
- **导入的团队智能体显示离线**：修复导入或克隆的智能体在列表中显示为离线、重启后才恢复在线的问题
- **扫描件遇非视觉模型退化复读**：修复非视觉模型读到纯扫描件 PDF 时进入无限重复输出的问题——现改为直接提示需切换到支持视觉的模型并结束本轮
- **列表状态指示灯不准确**：修复 Agent 正在用工具时（如 Read），聊天列表的状态灯仍显示绿色而非黄色的问题
- **委派多目标校验空缺**：修复 fan-out 委派不存在的目标时静默创建空壳子会话并显示「已完成」的问题；修复委派卡片对多目标显示为「unknown」的问题
- **AgentFS 热更新失效**：修复直接编辑 agent.json 文件后热更新不生效的问题（根因：文件监听器误将整个根目录判为隐藏而忽略）
- **默认模型路径 ProviderId 缺失/错配**：修复历史数据中缺失 providerId 导致默认模型永久漂移、错误 Provider 路由串台等问题
- **credentialSource 毒丸事故系统性修复**：将凭据源字段从硬枚举改为开放字符串，防止未来新凭据源类型导致老客户端算力功能全面瘫痪
- **glm 病态输出打死会话**：修复部分模型将工具参数错误输出为数组、经网关时报错“list object has no attribute items”导致整条会话不可恢复的崩溃
- **anthropic 兼容网关两类 400 自愈**：修复 deepseek-v4 等模型在第三方 anthropic 兼容网关上因 max_tokens 范围限制和 thinking 块回传缺失导致的 400 错误
- **关系图谱缺失「交付」边**：修复 Agent 交付结果的事件被静默丢弃、关系图谱永远画不出「交付」边的问题
- **send_message 异步回执错配**：修复并发调用 send_message 时异步回执可能被错误的同步等待方消费的问题
- **Windows 含空格 git 路径崩溃**：修复 Windows 上因 git 路径含空格导致所有 git 操作报错的问题
- **Windows「清除数据」失败**：修复只读 git 文件与保留设备名文件导致清除数据操作报 EPERM 失败的问题
- **点击 file:// 链接崩溃**：修复点击 Agent 回复中的本地文件链接导致 macOS 报错或 Windows 灰屏卡死的问题，现改为由系统默认应用打开
- **聊天输入框换行后光标不可见**：修复 Shift+回车换行后输入区超过高度上限时光标落在可视区之外、用户误以为换行无效的问题
- **消息区横向滚动条与长内容被裁剪**：修复长 URL、超宽表格等内容导致消息区出现横向滚动条或内容被裁剪不可见的问题，内容始终完整可见
- **消息操作栏盖住时间戳**：修复短消息时操作按钮溢出、盖住右侧时间戳的问题
- **多选模式下点击选择圆圈无响应**：修复聊天多选模式中点左侧选择圆圈偶尔无响应的问题
- **系统提示中 Windows 路径反斜杠丢失**：修复 Agent 复述 Windows 路径时反斜杠被 Markdown 转义吞噬的问题，改为以行内代码包裹
- **消息列表滑到底整窗闪烁**：修复触控板滚到消息列表底部后继续滑动导致整窗闪烁的问题
- **团队工作目录链路错误**：修复团队共享工作目录在委派和上下文注入中路径不正确的问题
- **codex off 档思考未显式关闭**：修复 Codex 模型选择关闭思考后仍可能发送推理参数的问题
- **话题详情消息条数显示异常**：修复话题详情中消息条数显示为亿级数值的问题
- **待审核窗口脏数据渲染崩溃**：修复 Agent 注册的部分服务条目字段缺失时导致待审核页面渲染崩溃的问题
- **compat 网关思考参数缺失**：修复未声明 thinkingFormat 的网关上选择思考档位后不发送任何开启参数的问题
- **市场仓库保存卡死**：修复添加仓库源并点保存后一直显示「保存中」且仓库显示两份的问题

## 改进
- **思考深度预算体系**：修复 UI 上 5 个深度档位实际只产生 3 个预算值、「深」与「最深」无法区分的问题；思考档位现在持久化至 agent.json，重启后保留选择；存量已固化的旧预算组自动识别忽略
- **动态输出预算与大输出一次成型**：输出 token 预算改为动态计算，不再被静态上限或手动分步续写限制，大输出可一次成型
- **上下文裁剪与 token 估算优化**：裁剪改为级联策略（截 tool_result→删老轮→中段折叠→突破软地板），消灭“截完仍超仍原样发出”的死锁；修复 token 估算中漏数 thinking/工具名/数组内容等问题，提升估算精度
- **Provider 并发限流加固**：Provider 级并发信号量增加有界队列、排队超时、可取消、热更新配置重试与诊断等能力，防止瞬时请求打爆上游
- **compat-proxy 按 ProviderId 精确路由**：代理层增加三层路由防线，按 ProviderId、入站 API Key 指纹匹配和按名兜底，防止多 Provider 同名模型串台
- **排队查询可中止**：排队中等待执行的查询现在支持用户手动停止或断线自动取消，不会继续消耗资源执行不可逆的工具调用；交互式查询优先于后台任务执行
- **Pipeline 组装阶段可取消**：查询在工具装配、上下文构建等准备阶段收到取消信号后立即中止，避免长时间阻塞后仍启动无意义的查询
- **Provider 信号量排队感知取消**：等待 Provider 并发槽位期间若请求被中止，立即从队列中移出，不再浪费排队时间
- **via-messaging 精确匹配回复**：委派/消息/协助请求增加 correlationId 关联字段，防止同一对话并发请求出现回复错配
- **子任务同步等待超时放宽至 30 分钟**：委派/spawn_agent/send_message 子任务超时从 5 分钟放宽为 30 分钟，超时不再自动中止任务，交由调用方判断是否继续等待或主动结束
- **Ls 工具默认免审批直接执行**：Ls（列出目录）作为只读安全工具改为默认直接执行，权限交由操作系统层管控，仅在权限拒绝时返回平台特定的授权引导文案
- **智能体配置密钥只写不读**：前端 API 响应中的密钥改为掩码显示（前 4 位 + •••），文件工具（Read/Grep/Glob）精确拒绝读取密钥文件，Bash 内置黑名单拦截含密钥文件名的命令
- **SSRF 防护下沉到逐 Agent**：SSRF 防护现在支持按 Agent 独立开关，即时生效；补齐 DNS Rebinding 防护，fetch 前锁定域名解析结果
- **standalone 鉴权与远程媒体访问加固**：统一 Agent Service 远程请求鉴权入口，原生媒体元素引入时限 HMAC 签名 URL，5 分钟有效期防长期 token 暴露
- **三层提示词体系**：全局/团队/Agent 三层叠加注入，集中式提示词中心支持可视化编辑预览，按全局/团队/agent/系统分组展示 systemPrompt 结构
- **算力页面显示预置配置版本号**：算力配置页面头部显示当前合并的 config-center 预设版本号，便于故障排查
- **Provider 级 requiredClientVersion 门控**：客户端版本低于 Provider 要求时，该 Provider 自动从模型选择器过滤，并显示需升级客户端的提示
- **算力配置韧性增强**：首装或损坏重建路径开启 salvage 防护，单坏 Provider 不再阻断整份配置加载或 Agent Service 启动
- **对话 PDF 导出改为客户端渲染**：导出 PDF 不再依赖服务端渲染，改为客户端 Electron printToPDF 渲染，为 Agent Service 独立部署解耦
- **独立部署数据外部化**：Agent Service 独立部署时全部 per-instance 数据路径统一跟随 DESIRECORE_HOME，TLS 证书不再落 bundle 目录
- **DESIRECORE_HOME 进程锁**：Agent Service 启动时获取数据根目录进程锁，多进程冲突时排队等待而非直接报错，防止多进程同时写同一数据目录
- **写路径原子化加固**：全仓覆盖写全部转为原子写入（临时文件+重命名），消除崩溃导致的文件半写损坏；agent.json 双锁体系统一为 per-agent 串行锁
- **导入导出规则单源化**：main 与 agent-service 双份相同的导入导出排除规则收敛为共享模块，消除规则漂移；剔除 legacy 死代码
- **运行时目录统一**：运行时工具链目录统一到 ~/.desirecore/runtime，告别不同环境分叉，老用户首次启动自动迁移
- **消息滚动条加宽命中区**：聊天消息区滚动条命中区域从 4px 扩至 10px，便于鼠标拖拽操作
- **关系热度格式化展示**：关系热度改为千分位+1 位小数展示（如 3,529.6），不再渲染长浮点撑破卡片布局
- **应用与服务页面增加手动刷新按钮**：应用与服务页面标题栏新增刷新按钮，可主动同步服务注册目录
- **服务目录状态实时更新**：服务注册状态变更时通过 Socket 实时推送前端，无需手动刷新
- **算力供应商公告与计费信息**：官方算力模型顶部展示「算力由合作伙伴 {供应商} 提供」公告；按 token/按次等计费模式展示价格与计价基准
- **任务板详情探测消除控制台噪音**：切换会话时的任务板探测请求不再返回 404 写入控制台，改为正常 200 探测语义
- **WorkflowRun 等待超时放宽**：WorkflowRun 工具默认不设工具层超时，可选参数精确控制等待时间，长工作流不再被提前判定为超时
- **委派提示词路径规则**：委派工具提示词新增路径规则说明，跨 Agent 传递资源须使用绝对路径
- **聊天消息列表滚动条防闪烁**：补充 overscroll-contain 防整窗闪烁，更新全局兜底规则
- **多选模式辅助功能改进**：选择圆圈改为语义化按钮，增加屏幕阅读器支持和键盘可达性
- **市场 pointer 条目升级门控**：pointer 条目的升级版本门槛字段正确展示，与内联条目行为一致
- **套餐卡片原价划线**：套餐价格支持展示原价划线和折扣百分比/限免标签
- **系统提示词精简去重**：自我认知提示词中移除重复的路径列举与僵尸能力声明，节省 token
