# DesireCore v10.0.57

**发布日期**: 2026-05-14

![v10.0.57 更新概览](/img/changelog/v10.0.57.png)

## 修复
- 修复了 Windows 上部分用户升级 DesireCore 时因系统安全扫描暂时占用程序文件而导致安装失败的问题，升级过程现在更稳定。
- 修复了某些 Windows 环境下安装器在结束旧版本进程时执行不完整的问题，减少了升级过程中提示失败或中断的情况。

## 改进
- 优化了 Windows 安装器的兼容性与容错处理，即使系统缺少部分 PowerShell 能力或命令执行异常，安装流程也能尽量继续完成。
